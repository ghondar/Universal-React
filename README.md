# Universal-React
Quick Example Universal React

## Setup
```shell
$ git clone https://github.com/ghondar/Universal-React.git
$ cd Universal-React
$ npm install
```

## Run

run webpack dev server

```shell
$ npm run webpack
```

and, in another terminal.

run server

```shell
$ npm run start-dev
```

## Build and Package

to build

```shell
$ npm run build
```
